Pertanyaan Umum

Apakah aman untuk akun TikTok Shop saya?

Sangat aman. OnTheTok tidak meminta password Seller Center. Kami menggunakan Chrome Extension yang hanya baca data, tidak modifikasi apapun di akun Anda.

Bagaimana jika kuota saya habis?

Bisa top-up add-on kapan saja atau upgrade paket instant. Tidak ada batasan untuk menambah kuota sesuai kebutuhan Anda.

Apakah support multiple SKU/varian produk?

Ya. Anda bisa input ratusan SKU sekaligus untuk tracking akurat. Sistem kami mendukung multi-SKU dan multi-campaign secara bersamaan.

Berapa lama setup awal?

Onboarding 30 menit. Guide ada di dalam app.Tim kami siap membantu jika ada kendala. Anda akan siap operasional dalam hari yang sama.

Apakah ada garansi?

Trial 7 hari gratis dengan akses penuh. Kami yakin Anda akan merasakan manfaatnya, detail hubungi admin.

Data kreator saya aman?

data kreator aman karena tersimpan di cloud, tidak perlu khawatir jika perangkat hilang.
